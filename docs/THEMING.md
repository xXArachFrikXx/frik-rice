# Theming System

This rice uses **matugen** to generate a full color palette from either a wallpaper
image or a hex accent color. The palette follows Material Design 3 (Google's design
language), which gives coherent colors across every app automatically.

---

## How It Works

```
You pick a preset or wallpaper
        ↓
matugen generates a palette (16+ colors: primary, secondary, surface, etc.)
        ↓
matugen templates render the palette into config files for each app:
  • ~/.config/waybar/colors.css       ← waybar colors
  • ~/.config/kitty/colors.conf       ← terminal colors
  • ~/.config/rofi/colors.rasi        ← launcher colors
  • ~/.config/cava/config             ← visualizer gradient
  • ~/.config/wlogout/style.css       ← logout screen
  • ~/.config/btop/themes/matugen.theme ← system monitor
  • ~/.config/hypr/Modules/colors.lua ← window borders
        ↓
Each app gets a SIGUSR signal to reload its colors live
(waybar, kitty, cava all reload without restart)
```

## Presets

Presets live in `~/.config/hypr/presets/`. Each `.preset` file has:

```bash
WALLPAPER=/home/Frik/Wallpapers/Snow2.jpg   # Wallpaper to set
SCHEME=tonal-spot                            # matugen color scheme type
SEED_COLOR=#f68500                           # Orange accent seed color
WAYBAR_THEME=solid-buttons                   # Which waybar style to use
```

**Press SUPER+W** to open the preset picker.

### Available matugen scheme types

| Scheme | Effect |
|--------|--------|
| `tonal-spot` | Moderate accent — surfaces stay neutral/dark, accent used sparingly. Best for subtle themes. |
| `vibrant` | High chroma — everything gets saturated with the accent color. More intense. |
| `expressive` | Vivid, varied colors across primary/secondary/tertiary. Colorful but coherent. |
| `fidelity` | Tries to match the source color as closely as possible. |
| `monochrome` | Single color family, no accent contrast. |
| `neutral` | Very muted, near-greyscale with a slight tint. |

### Adding a new preset

1. Create `~/.config/hypr/presets/my-theme.preset`
2. Fill in the fields (copy snow-winter.preset as a template)
3. Press SUPER+W → pick your new theme

If `SEED_COLOR` is set, matugen uses that hex color as the accent seed instead of
analyzing the wallpaper. This gives consistent, predictable colors regardless of what
the wallpaper image actually contains.

## Waybar Themes

Waybar has separate style files in `~/.config/waybar/style/`:

| Style | Look |
|-------|------|
| `solid-buttons.css` | Rounded pill-shaped module buttons (current default) |
| `islands.css` | Floating island groups |
| `full bar.css` | Traditional solid full-width bar |
| `sidebar.css` | Vertical sidebar layout |

Switch with **SUPER+SHIFT+W** (full theme picker) or **SUPER+CTRL+B** (style only).

---

## Files Auto-generated by matugen (do not edit manually)

These files are rebuilt every time you switch themes. Edits will be overwritten:

- `~/.config/hypr/Modules/colors.lua`
- `~/.config/hypr/colors.conf`
- `~/.config/waybar/colors.css`
- `~/.config/kitty/colors.conf`
- `~/.config/rofi/colors.rasi`
- `~/.config/cava/config`
- `~/.config/wlogout/style.css`
- `~/.config/btop/themes/matugen.theme`

To change how a color is applied, edit the **template** in
`~/.config/matugen/templates/` instead.
